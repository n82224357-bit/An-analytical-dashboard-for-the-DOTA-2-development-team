# ***Цель исследования:*** 
Разработать аналитический дашборд для отдела разработки DOTA 2, который позволит:
- оценивать масштаб игровой активности (число матчей, пользователей, участников);
- анализировать баланс фракций (Radiant / Dire) и типов атаки (Ranged / Melee);
- выявлять популярных персонажей и предметы, востребованные на старте игры;
- изучать распределение матчей по длительности и числу игроков в них;
- сравнивать ключевые характеристики персонажей (атака, скорость, восстановление, убийства, опыт в минуту) у победителей и проигравших в зависимости от ведущей роли;
- обеспечивать детализацию данных через селекторы для углублённого анализа отдельных фракций, исходов, персонажей и игроков.
Конечная цель дать разработчикам инструмент для принятия решений по балансировке игры: корректировке характеристик персонажей, оптимизации набора предметов, улучшению зрелищности и честности матчей.
# ***Что было сделано***
- Определена структура дашборда 
- Сформулированы ключевые метрики 
- Продумана интерактивность: добавлены селекторы , обеспечена детализация от общих показателей до конкретных матчей и игроков.
- Подготовлены данные для визуализации: загружены CSV файлы в DataLens (предварительно осуществлена предобработка данных в Google-таблицах); настроены соединения между таблицами, созданы вычисленные поля для ключевых метрик
